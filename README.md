# ClassSync Planner

Xcode 12.5 / iOS 14 호환용 SwiftUI 프로젝트입니다.

이번 버전 반영사항:
- 앱 실행 시 바로 Eclass 로그인 화면으로 가지 않고 자체 시작 화면 표시
- Eclass 로그인 버튼을 누르면 공식 Eclass 로그인 WebView 표시
- 로그인 정보는 앱에 저장하지 않고 비영구 WebView 세션 사용
- 로딩 화면을 카드형 UI로 개선
- 메인 화면을 대시보드 / 강의 탭 구조로 변경
- 하단 바와 오늘 할 일 영역 제거
- 대시보드에는 최근 공지 5개만 표시
- 강의 탭에는 과목별 카드 목록 표시
- 과목 상세 화면에 과제 / 공지 / 동영상 탭 추가
- 공지사항 게시판만 수집하도록 수정하여 Q&A 게시판 혼합 방지
- 공지 상세 내용 확인 화면 추가
- D+숫자 대신 마감 지남 표시
- 과목별 컬러 배지 적용

주의:
- 학교 Eclass HTML 구조가 바뀌면 파싱 결과가 달라질 수 있습니다.
- Xcode 12.5 기준으로 최신 SwiftUI 문법을 피해서 작성했습니다.
